# Portfolio Site

Next.js와 Tailwind CSS로 만든 개인 포트폴리오 사이트입니다.

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

http://localhost:3000 에서 확인할 수 있습니다.

## Vercel 배포하기

1. GitHub에 코드 푸시
2. https://vercel.com 에서 로그인
3. "New Project" 클릭
4. GitHub 저장소 선택
5. 배포 버튼 클릭

## 기술 스택

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
