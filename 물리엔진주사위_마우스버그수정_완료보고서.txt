## 물리엔진 주사위 마우스 버그 수정 완료 보고서

작업 일시: 2025-01-18
작업 내용: 주사위 던지기 높이 조정 및 마우스 이벤트 버그 수정

### 수정된 문제:

#### 1. 주사위 던지기 높이 조정
- **시작 높이**: 10 + i*2 → 5 + i*1.5 (50% 감소)
- **초기 속도**: 
  - 수직: 10~20 → 5~10 (50% 감소)
  - 수평: ±20 → ±10 (50% 감소)
- **각속도**: ±10 → ±5 (50% 감소)
- **벽 높이**: 15 → 20 (주사위 이탈 방지)

#### 2. 마우스 이벤트 버그 수정
- **문제**: 마우스 클릭 후 놓아도 카메라가 계속 따라다님
- **원인**: 이벤트 리스너가 제대로 제거되지 않음
- **해결**:
  - isMouseDown 플래그 추가
  - 이벤트 핸들러 바인딩 저장
  - document 레벨 이벤트 리스너 정확한 제거
  - 휠 클릭(가운데 버튼) 무시

### 코드 개선 사항:
```javascript
// 마우스 상태 추적
this.isMouseDown = false;

// 이벤트 핸들러 저장
this.boundMouseMove = this.onMouseMove.bind(this);
this.boundMouseUp = this.onMouseUp.bind(this);

// 왼쪽 버튼만 처리
if (event.button !== 0) return;

// 정확한 이벤트 제거
document.removeEventListener('mousemove', this.boundMouseMove);
```

### 테스트 결과:
- ✅ 주사위가 벽 밖으로 나가지 않음
- ✅ 마우스 드래그 후 정상적으로 해제됨
- ✅ 휠 클릭이 카메라 회전을 트리거하지 않음
- ✅ 적절한 던지기 높이와 속도
