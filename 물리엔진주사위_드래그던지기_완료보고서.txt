## 물리엔진 주사위 드래그 던지기 기능 추가 완료 보고서

작업 일시: 2025-01-18
작업 내용: 마우스 드래그로 주사위 던지기 방향과 세기 조절 기능 구현

### 추가된 기능:

#### 1. 드래그 던지기 시스템
- ✅ **오른쪽 클릭 + 드래그**로 던지기 방향과 세기 결정
- ✅ 드래그 시 카메라 회전 비활성화 (충돌 방지)
- ✅ 최소 드래그 거리 20px 설정
- ✅ 드래그 방향이 주사위 던지기 방향에 반영

#### 2. 시각적 피드백
- ✅ **3D 화살표** 표시
  - 드래그 방향 실시간 표시
  - 파워에 따라 크기 변경
  - 색상 변화 (녹색→빨간색)
- ✅ **파워 게이지** UI
  - 화면 중앙에 표시
  - 0~100% 파워 표시
  - 그라데이션 바로 시각화

#### 3. 사용자 경험 개선
- ✅ 직관적인 조작 방법
- ✅ 명확한 시각적 피드백
- ✅ 기존 버튼 던지기와 병행 가능
- ✅ 사용 안내 메시지 추가

### 구현 세부사항:

```javascript
// 드래그 파워 계산
const power = Math.min(distance / 200, 1); // 최대 200px 드래그 = 100% 파워

// 방향 벡터 계산
body.velocity.set(
    -deltaX * 0.1 * power,  // X 방향 (화면 좌우)
    Math.random() * 10 + 20, // Y 방향 (높이)
    deltaY * 0.1 * power     // Z 방향 (화면 상하)
);
```

### 조작 방법:
1. **오른쪽 마우스 버튼** 클릭하고 누른 상태 유지
2. 원하는 방향으로 드래그 (화살표와 게이지 확인)
3. 마우스 버튼을 놓으면 주사위 던지기

### 시각적 요소:
- **화살표**: 네온 그린 → 빨간색 (파워 증가)
- **파워 게이지**: 0~100% 표시
- **방향 표시**: 3D 화살표가 드래그 방향 지시
